# chromedictator


 Install the most recent stable version of Go from https://golang.org/dl/


    git clone https://github.com/stts-se/chromedictator
    cd chromedictator
    go get

To start the server:

     go run chromedictator.go

or
     go build
     ./chromedictator


Go to the following URL in Google Chrome: 

     http://localhost:7654
